Details

First of all you need to download the application from the Home page or from this link. Next unzip the contents of the zip file (keeping the zip hierarchy) to a folder of your choice. Unblock AudioDownloader.exe file and run it by double clicking. If you feel you need more information on how to run the Audio Downloader for the first time please follow this link.

Application overview

Main application components:

  1. The YouTube link goes into this text box
  2. Audio quality
  3. Press to download the file
  4. Status of the application
  5. Progress bar
  6. Opens audio folder
  7. Opens this web page

Use case

Download a file:

  1. Paste YouTube link into text box
  2. Select audio quality
  3. Press download
  4. Open audio folder
