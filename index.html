Mp3 downloader

Features:

  • Downloads audio from YouTube
  • Easy to use
  • Clean interface
  • Main window always on top
  • Extracts audio in best quality
  • Before converting to mp3 allows to select the desired audio quality
  • Displays current download speed
  • Uses Youtube-dl for downloading and FFmpeg for processing
  • Allows usage of Youtube-dl command line interface
  • Preserves raw audio files (option)
  • Can store information about the download in Google Sheets (option)
  • More features still to come :)
